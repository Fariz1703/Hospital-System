# 🏥 Hospital Management System (Odoo 17 Custom Module)

A simple **Hospital Management System** built using **Odoo 17** with custom modules to manage basic hospital operations such as patient registration, appointment scheduling, doctor assessments, prescriptions, and payment processing.

This project demonstrates the implementation of **custom Odoo modules using Python** to support healthcare workflow management.

---

# 🚀 Features

* **Patient Registration**
  Register and manage patient information.

* **Appointment Management**
  Create and manage patient appointments with doctors.

* **Queue Ticketing System**
  Generate queue tickets for patients during check-in.
  Patients receive a small queue ticket PDF containing their queue number, doctor, and appointment details.

* **Check-in Queue System**
  Manage patient check-in and waiting queue.

* **Doctor Assessment**
  Doctors can record patient assessments including medical information.

* **Prescription Management**
  Generate and manage prescriptions for patients.

* **Payment Processing**
  Handle patient payments after consultation and treatment.
  
* **Invoice Report**
  Generate PDF invoices for patient services including consultation and additional treatments.

# ⚙️ Installation

1. Clone this repository

```bash
git clone https://github.com/yourusername/hospital-management-odoo.git
```

2. Move the module into your **Odoo addons folder**

```
odoo/custom_addons/
```

3. Restart the Odoo server.

4. Activate **Developer Mode** in Odoo.

5. Go to:

```
Apps → Update Apps List
```

6. Search for:

```
Hospital-System
```

7. Click **Install**.
